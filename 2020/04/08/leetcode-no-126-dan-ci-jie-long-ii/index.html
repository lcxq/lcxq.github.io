单词接龙Ⅱ

给定两个单词（beginWord 和 endWord）和一个字典 wordList，找出所有从 beginWord 到 endWord 的最短转换序列。转换需遵循如下规则：

  1. 每次转换只能改变一个字母。
  2. 转换过程中的中间单词必须是字典中的单词。

说明:

  • 如果不存在这样的转换序列，返回一个空列表。
  • 所有单词具有相同的长度。
  • 所有单词只由小写字母组成。
  • 字典中不存在重复的单词。
  • 你可以假设 beginWord 和 endWord 是非空的，且二者不相同。

示例 1:

输入:
beginWord = "hit",
endWord = "cog",
wordList = ["hot","dot","dog","lot","log","cog"]

输出:
[
  ["hit","hot","dot","dog","cog"],
  ["hit","hot","lot","log","cog"]
]

示例 2:

输入:
beginWord = "hit"
endWord = "cog"
wordList = ["hot","dot","dog","lot","log"]

输出: []

解释: endWord "cog" 不在字典中，所以不存在符合要求的转换序列。

方法一

自己只能用dfs写一个过了19个case的代码。

class Solution {
    public List<List<String>> findLadders(String beginWord, String endWord, List<String> wordList) {
        List<List<String>> ans = new ArrayList<>();
        boolean[] used = new boolean[wordList.size()];
        List<String> tempList = new ArrayList<>();
        tempList.add(beginWord);
        dfs(ans, tempList, wordList, used, endWord);
        return ans;
    }
    private void dfs(List<List<String>> ans, List<String> tempList, List<String> wordList, boolean[] used, String endWord) {
        if (!ans.isEmpty() && tempList.size() > ans.get(0).size()) return; 
        if (tempList.get(tempList.size() - 1).equals(endWord)) {
            if (ans.isEmpty() || ans.get(0).size() == tempList.size()) {
                ans.add(new ArrayList<>(tempList));
            } else if (ans.get(0).size() > tempList.size()) {
                ans.clear();
                ans.add(new ArrayList<>(tempList));
            }
            return;
        }
        for (int i = 0; i < used.length; i++) {
            if (!used[i] && helpFunc(tempList.get(tempList.size() - 1), wordList.get(i))) {
                tempList.add(wordList.get(i));
                used[i] = true;
                dfs(ans, tempList, wordList, used, endWord);
                used[i] = false;
                tempList.remove(tempList.size() - 1);
            } 
        }
    }
    private boolean helpFunc(String str1, String str2) {
        int sum = 0;
        for (int i = 0; i < str1.length(); i++) {
            if (str1.charAt(i) != str2.charAt(i)) {
                sum ++;
            }
        }
        if (sum <= 1) return true;
        else return false;
    }
}
