环境配置

所具备的环境配置

  • Tomcat
  • JDK12
  • Maven
  • Spring5
  • SpringMVC
  • MyBatis

所用IDE

  • IDEA

创建Webapp项目

  1. IDEA新建项目，选择Maven项目，勾选Create from archetype，选择maven-archetype-webapp
  2. GroupId: com.Lcxq; ArtifactId: SSMLoginDemo
  3. 在maven配置界面选择本地安装的maven目录，并且将setting.xml改为本地maven/conf目录下的setting.xml文件，这样才会使用阿里云镜像实现加速
  4. 最后修改一下需要的项目路径即可，等待maven的自动拉取

创建目录结构

  1. 在main目录下创建java目录，并标记为Sources Root
  2. 在main目录下创建resources目录，并标记为Resources Root
  3. 在根目录下创建target目录，idea将自动识别为Excluded
  4. 在java目录下创建com包，在com包中创建controller,dao,entity,service四个包
  5. 在resources目录下创建mapper包和spring包，以及jdbc.properties文件，log4j.properties文件，mybatis-config.xml文件，用于配置
  6. （如有需要）在src目录下创建test文件夹，并创建java和resources文件夹，并将它们分别标记为Test Resources Root和Test Resources Root

各种配置文件

需要配置的文件主要在resources目录下

mapper

新建文件UserDao.xml作为mybatis映射文件

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper
        PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="com.dao.UserDao">
    <select id="findUser" resultType="User" parameterType="String">
        SELECT *
        FROM t_user
        WHERE username=#{username}
    </select>

    <insert id="addUser">
        INSERT INTO t_user(username, pwd)
        VALUES (#{username}, #{pwd})
    </insert>
</mapper>

本demo中就用到根据用户名查找username以及新增user两个SQL

< mapper >标签中的namespace设置为dao目录下创建的UserDao接口（待会写dao包时会创建）

< select >与< insert >标签下的id名需要与UserDao接口中的方法一致

spring

  1. spring-dao.xml

负责dao层的相关配置，包括数据库，连接池，SqlSessionFactory，dao接口扫描等

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:context="http://www.springframework.org/schema/context"
       xsi:schemaLocation="http://www.springframework.org/schema/beans
    http://www.springframework.org/schema/beans/spring-beans.xsd
    http://www.springframework.org/schema/context
    http://www.springframework.org/schema/context/spring-context.xsd">
    <!-- 配置整合mybatis过程 -->
    <!-- 1.配置数据库相关参数properties的属性：${url} -->
    <context:property-placeholder location="classpath:jdbc.properties" />

    <!-- 2.数据库连接池 -->
    <bean id="dataSource" class="com.mchange.v2.c3p0.ComboPooledDataSource">
        <!-- 配置连接池属性 -->
        <property name="driverClass" value="${jdbc.driver}" />
        <property name="jdbcUrl" value="${jdbc.url}" />
        <property name="user" value="${jdbc.username}" />
        <property name="password" value="${jdbc.password}" />

        <!-- c3p0连接池的私有属性 -->
        <property name="maxPoolSize" value="30" />
        <property name="minPoolSize" value="10" />
        <!-- 关闭连接后不自动commit -->
        <property name="autoCommitOnClose" value="false" />
        <!-- 获取连接超时时间 -->
        <property name="checkoutTimeout" value="10000" />
        <!-- 当获取连接失败重试次数 -->
        <property name="acquireRetryAttempts" value="2" />
    </bean>

    <!-- 3.配置SqlSessionFactory对象 -->
    <bean id="sqlSessionFactory" class="org.mybatis.spring.SqlSessionFactoryBean">
        <!-- 注入数据库连接池 -->
        <property name="dataSource" ref="dataSource" />
        <!-- 配置MyBaties全局配置文件:mybatis-config.xml -->
        <property name="configLocation" value="classpath:mybatis-config.xml" />
        <!-- 扫描entity包 使用别名 -->
        <property name="typeAliasesPackage" value="com.entity" />
        <!-- 扫描sql配置文件:mapper需要的xml文件 -->
        <property name="mapperLocations" value="classpath:mapper/*.xml" />
    </bean>

    <!-- 4.配置扫描Dao接口包，动态实现Dao接口，注入到spring容器中 -->
    <bean class="org.mybatis.spring.mapper.MapperScannerConfigurer">
        <!-- 注入sqlSessionFactory -->
        <property name="sqlSessionFactoryBeanName" value="sqlSessionFactory" />
        <!-- 给出需要扫描Dao接口包 -->
        <property name="basePackage" value="com.dao" />
    </bean>
</beans>

service层

创建UserService接口，其中包含两个业务方法，分别为

  • findUserAndSet(寻找用户，找不到就加入)
  • findUserAndMatchPwd(寻找用户，找到了就继续匹配密码)
package com.service;

import com.entity.User;

public interface UserService {
    /**
     *
     * @param user
     * @return
     */
    String findUserAndSet(User user);

    /**
     *
     * @param user
     * @return
     */
    String findUserAndMatchPwd(User user);
}

在impl目录下编写接口实现类

package com.service.impl;

import com.dao.UserDao;
import com.entity.User;
import com.service.UserService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

@Service
public class UserServiceImpl implements UserService {
    @Autowired
    private UserDao userDao;

    @Override
    public String findUserAndSet(User user) {
        System.out.println(userDao);
        User findUserResult = userDao.findUser(user.getUsername());
        if (findUserResult != null) return "User Exists";
        else userDao.addUser(user);
        return "Succeed Adding User";
    }

    @Override
    public String findUserAndMatchPwd(User user) {
        User findUserResult = userDao.findUser(user.getUsername());
        if (findUserResult == null) return "User Not Exists";
        else {
            if (findUserResult.getPwd().equals(user.getPwd())) {
                return "Succeed Matching Pwd";
            } else {
                return "Fail to Match Pwd";
            }
        }
    }
}

controller层

controller目录下新建UserController类

package com.controller;

import com.Utils.SHA256;
import com.entity.User;
import com.service.UserService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.Map;

@RestController
public class UserController {
    @Autowired
    private UserService userService;
    private User init(Map<String, String> map) {
        User user = new User();
        user.setUsername(map.get("username"));
        user.setPwd(SHA256.str2SHA256(map.get("pwd")));
        return user;
    }
    @ResponseBody
    @RequestMapping(value = "/findUserAndSet", method = RequestMethod.POST)
    public String findUserAndSet(@RequestBody Map<String, String> map) {
        User user = init(map);
        return userService.findUserAndSet(user);
    }

    @ResponseBody
    @RequestMapping(value = "/findUserAndMatchPwd", method = RequestMethod.POST)
    public String findUserAndMatchPwd(@RequestBody Map<String, String> map) {
        User user = init(map);
        return userService.findUserAndMatchPwd(user);
    }
}

实现两个api，与service层中的方法对应。

由于需要将密码转换为SHA-256，因此在com目录下新建Utils目录，并创建SHA256类，实现转换方法。

package com.Utils;

import java.math.BigInteger;
import java.security.MessageDigest;

public class SHA256 {
    static public String str2SHA256(String str){
        byte[] result = new byte[256];
        try {
            MessageDigest md = MessageDigest.getInstance("SHA-256");
            md.update(str.getBytes("UTF-8"));
            result = md.digest();
        } catch (Exception e) {
            e.printStackTrace();
        }
        return new BigInteger(1, result).toString(16);
    }
}

项目启动(Tomcat)

至此，所有代码已经编写完成，下面需要用Tomcat进行启动。

  1. 在IDEA的运行栏中点击编辑配置 -> 左上角+号 -> Tomcat Server/Local
  2. Application server中选择自己本地安装的Tomcat（需要点击右侧Configure）
  3. 在Deplotment标签页下点击右侧+号/Artifact，选择war explored
  4. 点击确定，运行，项目启动完成

也可以用maven的tomcat插件进行启动Tomcat
