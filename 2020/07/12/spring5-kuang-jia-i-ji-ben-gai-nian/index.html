基本概念

什么是Spring框架？

  1. Spring是轻量级的开源的J2EE开发框架
  2. Spring可以解决企业应用开发的复杂性
  3. Spring有两个核心部分：IOC和Aop

IOC: 控制反转，把创建对象过程交给Spring进行管理。

Aop：面向切面，不修改源代码的情况下进行功能增强

有什么特点？

  1. 方便解耦，简化开发
  2. Aop编程支持
  3. 方便测试，整合Junit
  4. 方便集成各种优秀框架
  5. 降低API使用难度（例如JDBC）
  6. 方便进行事务操作

入门案例

下载Spring框架，新建项目

先从这里下载Spring框架

解压缩后新建项目，导入jar包

在入门案例中，我们导入5个基本的jar包分别是beans;context;core;expression;commons-logging

先在项目根目录中创建一个lib文件夹，将这五个jar包复制到里面，然后在IDEA的 文件->项目结构->依赖->右侧的加号->JARs或目录 中将lib里的五个jar包选中加入。

然后就可以开始写代码了。

创建普通类、在这个类中创建普通方法

在项目根目录中新建一个名为com.atguigu.spring5的包，在其中新建User类文件，并添加如下代码。

package com.atguigu.spring5;

public class User {
    public void add() {
        System.out.println("add.....");
    }
}

常规来说如果我们要用User类中的add方法的话需要User user = new User(); 然后 user.add(); 现在我们尝试用Spring来创建和调用。

创建 Spring 配置文件，在配置文件配置创建的对象

  1. Spring配置文件使用xml格式

在src目录中创建bean1.xml文件，选择Spring Config，文件中会自带如下代码

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">

</beans>
  1. 在中间加入bean标签配置，id为另外给类取的名，class为包+类名（全路径）
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">

    <!--配置User对象创建-->
    <bean id="user" class="com.atguigu.spring5.User"></bean>
</beans>

进行测试代码编写

在com.atguigu.spring5目录中新建testdemo目录，在其中新建TestSpring5目录，加入如下代码

package com.atguigu.spring5.textdemo;

import com.atguigu.spring5.User;
import org.junit.Test;
import org.springframework.context.ApplicationContext;
import org.springframework.context.support.ClassPathXmlApplicationContext;

public class TestSpring5 {

    @Test
    public void testAdd() {
        //1 加载Spring配置文件
        ApplicationContext context =
                new ClassPathXmlApplicationContext("bean1.xml");
        //2 获取配置创建的对象
        User user = context.getBean("user", User.class);

        System.out.println(user);
        user.add();
    }
}

入门案例总结

  1. 下载Spring5
  2. 导入Jar包依赖
  3. 创建普通类以及方法
  4. 创建xml配置文件，添加bean标签实现类的配置
  5. 测试代码中加载配置文件，获取配置创建的对象，实现方法的调用
