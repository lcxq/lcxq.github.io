JDBCTemplate

什么是JDBCTemplate

Spring框架对JDBC进行封装，使用JDBCTemplate方便实现对数据库操作

准备工作

（1）引入jar包

需要添加druid、mysql、jdbc、orm、tx五个jar包

（2）在spring配置文件配置数据库连接池

<!-- 数据库连接池 -->
    <bean id="dataSource" class="com.alibaba.druid.pool.DruidDataSource"
          destroy-method="close">
        <property name="url" value="jdbc:mysql:///user_db" />
        <property name="username" value="root" />
        <property name="password" value="admin" />
        <property name="driverClassName" value="com.mysql.jdbc.Driver" />
    </bean>

（3）配置JDBCTemplate对象，注入DataSource

<!--    JDBCTemplate对象-->
<bean id="jdbcTemplate" class="org.springframework.jdbc.core.JdbcTemplate">
<!--        注入DataSource对象-->
    <property name="dataSource" ref="dataSource"></property>
</bean>

dataSource使用set方法注入

（4）创建service类和dao类，在dao注入jdbcTemplate对象

<!--    组件扫描-->
<context:component-scan base-package="com.atguigu"></context:component-scan>
  • Service
@Service
public class BookService {
    //注入Dao
    @Autowired
    private BookDao bookDao;
}
  • Dao
@Repository
public class BookDaoImpl implements BookDao{
    //注入JDBCTemplate
    @Autowired
    private JdbcTemplate jdbcTemplate;
}

  1. 编写service和dao

（1）在dao进行数据库添加
（2）调用jdbcTemplate对象里面update方法实现添加操作

  • 有两个参数
  • 第一个参数：sql语句
  • 第二个参数：可变参数，设置sql语句值
@Repository
public class BookDaoImpl implements BookDao{

    //注入JDBCTemplate
    @Autowired
    private JdbcTemplate jdbcTemplate;

    @Override
    public void add(Book book) {
        //1 创建sql语句
        String sql = "insert into t_book values(?,?,?)";
        //2 调用方法实现
        Object[] args = {book.getUserId(), book.getUsername(), book.getUstatus()};
        int update = jdbcTemplate.update(sql, args);
        System.out.println(update);
    }
}

这里在测试过程中报了这个错

Unknown initial character set index '255' received from server.

查了之后似乎是编码集的问题，在jdbc配置数据库改为

<property name="url" value="jdbc:mysql:///user_db?characterEncoding=utf-8"/>

即可解决。

写法和添加基本一致，只是sql语句不同而已。

修改：

update t_book set username=?, ustatus=? where user_id=?;

删除

delete from t_book where user_id=?;

查询

  1. 返回某个值

使用queryForPbject方法

  • 有两个参数
  • 第一个参数：sql语句
  • 第二个参数：返回类型Class
@Override
public int selectCount() {
    String sql = "select COUNT(*) from t_book";
    Integer count = jdbcTemplate.queryForObject(sql, Integer.class);
    return count;
}
  1. 返回对象

场景：查询图书详情

同样使用queryForObject方法，但是传入三个参数

  • sql语句
  • RowMaper：RowMaper是一个接口，返回不同类型数据，使用这个接口里面实现类完成数据封装。
  • sql语句值
@Override
public Book findBookInfo(String id) {
    String sql = "select * from t_book where user_id=?";
    //调用方法
    Book book = jdbcTemplate.queryForObject(sql, new BeanPropertyRowMapper<Book>(Book.class), id);
    return book;
}
  1. 返回集合

场景：查询图书列表分页……

使用方法query，参数同queryForObject

@Override
public List<Book> findAllBook() {
  String sql = "select * from t_book";
  List<Book> list = jdbcTemplate.query(sql, new BeanPropertyRowMapper<Book>(Book.class));
  return list;
}

批量添加

  1. 批量操作：操作表里面多条记录
  2. JdbcTemplate实现批量添加操作

使用方法batchUpdate

  • sql语句
  • List集合，添加多条记录数据
@Override
public void batchAddBook(List<Object[]> batchArgs) {
    String sql = "insert into t_book values(?,?,?)";
    int[] ints =  jdbcTemplate.batchUpdate(sql, batchArgs);
    System.out.println(Arrays.toString(ints));
}
